# Structured Brain

This is the durable knowledge layer. Use PARA:

- `projects/`: active initiatives with a clear end state
- `areas/`: ongoing responsibilities without a fixed end date
- `resources/`: reusable references, standards, and research
- `archives/`: inactive or superseded material kept for auditability

Rules:
- keep each file narrow in scope
- put the freshest summary near the top
- use links instead of duplication
- mark superseded facts explicitly before archiving
